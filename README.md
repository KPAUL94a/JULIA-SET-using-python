# JULIA-SET-using-python
A Julia set is a type of intricate, infinitely complex fractal pattern created by iterating a simple mathematical formula on the complex plane. Discovered by French mathematician Gaston Julia, these shapes illustrate chaos theory and are closely tied to the famous Mandelbrot set.

Python libraries used in this are numpy and matplotlib
